# 競プロメモ

- vectorにcontainsはない。setとかにある。
- 範囲for文を活用する。
- for文のインデックスは複数あると間違えることがあるしデバッグも辛いので、ちゃんと命名する
- いもす法ではブロックの始まりで+、終わりの次（つまり次のブロックの始まり）で-する。
- いもす法では終わりの次で-する関係で、配列のサイズを1多くとる。
- 二次元累積和で、例えば縦に累積和を取るときは、for文の横の変数は-1しなくて良い。
- 飛び飛び値とキーを管理したい時はvectorではなくmap
- a < b < c の時に、bを固定して考えると2重ループが2回の1重ループになるかも
- 周期がmの時、nの次の値は(n + 1) % mとすると綺麗。
- 年月の隣接判定は通算月数の差が1かどうか
- 木は二部グラフ
- 二部グラフは隣り合う頂点を二色で塗りわけられる
- 有向グラフの強連結成分分解はACL
